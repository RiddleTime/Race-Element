SetupLinks

If you are running Race Element 1.1 or higher you will be able to to create SetupLinks in the Setup Tab. When clicked, these SetupLinks open up the setup importer for race element. You are able to create them for discord as well, basically for markdown.

For discord such a link would look like this: SetupLink: s1 Bentley Continental GT3 2018 - nurburgring
